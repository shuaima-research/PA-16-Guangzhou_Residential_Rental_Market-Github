# Empirical Analysis of Guangzhou Residential Rental Markets

This repository contains the analysis scripts and derived outputs for a
September 2021 study of residential rental markets in Guangzhou, China.

The study uses 3,000 publicly accessible online rental listings collected
and cleaned by the author. Hedonic regression models are used to examine
the relationships between asking rents, dwelling characteristics, and
district location.

## Repository structure

- `scripts/`: R scripts for regression, diagnostics, principal-component
  analysis, and clustering.
- `outputs/figures/`: Figures generated by the analysis.
- `outputs/tables/`: Aggregated descriptive statistics and model results.

## Data and paper

The underlying rental-listing data and the accompanying paper are not
included in this public repository. The data and paper are available from
the author upon reasonable request.
